# koban-cli

The `koban` command-line tool for [Invoice Ninja](https://invoiceninja.com),
built for humans and AI agents.

Install with `cargo install koban-cli` (installs a `koban` binary) or via the
project [README](../../README.md) instructions. The reusable API client lives in
the [`koban`](https://crates.io/crates/koban) library crate.
